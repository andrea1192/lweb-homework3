Questo minisito contiene parte dei miei appunti per il primo modulo del corso "Linguaggi per il Web" (A.A. 2023/2024), trascritti in un formato più discorsivo ed aumentati con informazioni raccolte dalla documentazione di Mozilla. I contenuti sono navigabili utilizzando il menu a sinistra.

Funzionamento del sito

Ciascuna pagina è generata dinamicamente da uno script PHP, che carica il contenuto richiesto da un repository di documenti XML.

Il repository è inizializzato dallo script install.php, che preleva una lista di contenuti di esempio, li converte in XML e valida il risultato secondo un DTD oppure uno schema a seconda dell'opzione scelta.
