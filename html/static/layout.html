Layout

Oltre alle modalità di allineamento con gli elementi circostanti descritte in Box model, la proprietà display può controllare il posizionamento degli elementi all'interno di ciascun box.

Normal flow

Per impostazione predefinita, il contenuto di un box è posizionato secondo il cosiddetto normal flow: come già descritto, gli elementi di tipo block riempiono orizzontalmente il box, crescendo verticalmente per accomodare il contenuto, mentre gli elementi inline non hanno dimensioni proprie e si adattano esclusivamente al proprio contenuto.

Essendo l'impostazione predefinita, gran parte del layout di questo minisito è visualizzato in normal flow. In particolare, si è scelto di lasciare che i blocchi header, wrapper e footer si disponessero verticalmente uno dopo l'altro come previsto da questo modello.

Per cambiare questo comportamento, è possibile agire in due modi:

  • alterare la posizione predefinita dei box interni, utilizzando la proprietà position
  • cambiare completamente il tipo di layout applicato, agendo sull'attributo display del contenitore

Layout alternativi

display: flex

Questa impostazione attiva il layout di tipo Flexbox, trasformando il box in un flex container e gli elementi contenuti in flex items.

Flexbox è un tipo di layout pensato per disporre gli elementi su una dimensione, l'asse principale, specificato con flex-direction. Un utilizzo tipico è suddividere lo spazio disponibile su questo asse in parti uguali:


.item1 { flex: 1; }

.item2 { flex: 1; }

Indipendentemente dall'impostazione corrente di box-sizing, lo spazio disponibile si riferisce a quello residuo dopo l'applicazione di bordi e padding.

Flexbox è stato utilizzato per disporre orizzontalmente i blocchi interni alle sezioni header, wrapper e footer. Ad esempio, il wrapper centrale è composto di due flex-item che costituiscono la colonna del menu, di una larghezza fissata a 150px, e quella del contenuto, che occupa il resto dello spazio.

Per ottenere un layout a tre colonne sarebbe stato sufficiente aggiungere una terza flex-item all'interno dello stesso container.

display: grid

CSS grid layout organizza gli elementi su due dimensioni, in righe e colonne, similarmente a quanto avveniva con le tabelle. Gli elementi all'interno prendono il nome di grid items.

Righe e colonne possono essere create con grid-template-rows e grid-template-columns, specificando le loro dimensioni. Ad esempio, per ottenere un layout a tre colonne:

grid-template-columns: 15% 70% 15%

Se colonne occupano complessivamente il 100% della larghezza del contenitore, l'aggiunta di un gap tra di loro potrebbe provocare un overflow nelle regioni circostanti. Un modo per evitarlo è l'utilizzo dell'unità di misura fr, che divide solo lo spazio effettivamente disponibile dopo l'applicazione di bordi e margini: 

grid-template-columns: 15fr 70fr 15fr

L'altra dimensione si adatta automaticamente al contenuto, oppure può essere specificata esplicitamente con grid-auto-columns e grid-auto-rows:

grid-auto-rows: 100px

Una volta creata la griglia desiderata, gli elementi possono essere posizionati utilizzando le proprietà grid-column e grid-row:


.item1 { grid-column: 1; }

.item2 { grid-column: 2 / 3; }
