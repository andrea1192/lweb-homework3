Box model

Gli elementi HTML vengono presentati al client racchiusi in box. Le proprietà del box nel quale è racchiuso un elemento, modificabili con CSS, determinano come questo verrà riprodotto dal client.

Tipi di box

Esistono due tipologie fondamentali di box: block ed inline. Ogni elemento appartiene per specifica ad una di queste tipologie, ma è possibile scegliere di cambiarla utilizzando l'attributo display.

block

Si estendono orizzontalmente su una riga dedicata, utilizzando l'intera larghezza del box nel quale sono rinchiusi. Tutte le proprietà geometriche possono essere modificate e verranno rispettate.

inline

Utilizzano la riga nel quale sono stati aperti, allargandosi solo quanto basta per il loro contenuto. Alcune proprietà geometriche, come width ed height, non vengono rispettate.

inline-block

Può essere inteso come compromesso tra block ed inline: come inline, utilizza la riga nel quale è aperto. A differenza di inline, consente di impostare esplicitamente width ed height.

Si è scelto di modificare il tipo di box in due istanze:

  • Nel menu laterale, per trasformare i link (<a>) da inline a block: in questo modo occupano interamente l'interno dell'elemento che li contiene, e la superficie cliccabile aumenta di conseguenza.
  • Nell'header di questi blocchi di annotazioni: in questo modo è stato possibile impostare un margine per il contenuto (content) dello pseudo-elemento ::before.

Parti di un box

Ogni box ha quattro aree fondamentali:

  • margin è la parte più esterna del box, le cui dimensioni determinano la distanza del box dai box circostanti. Se anche questi dispongono di un margine non nullo, quest'area può collassare automaticamente alla misura maggiore tra le due.
  • border è la linea che delimita il box.
  • padding è la zona che separa il bordo del box dal suo contenuto.
  • content è la zona nel quale il contenuto è visualizzato.

Contrariamente a quanto si potrebbe pensare, di default width ed height impostano la dimensione del solo contenuto, al quale poi andranno sommate le misure delle aree circostanti per ottenere le dimensioni effettive del box.

È possibile cambiare questo comportamento con:

box-sizing: border-box

In questo modo, le misure si riferiranno all'area delimitata dal bordo.
